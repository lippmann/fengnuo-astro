# 恐怖组织为何喜欢认领恐袭

**作者：冯诺**


> 近年来，每逢恐怖袭击或重大灾难，都有恐怖分子忙不迭宣称是他们所为。恐怖分子为何总喜欢将这些事情揽到自己身上呢？其背后，有炫耀实力、行动宣传、同行竞争等因素。


![](http://img3.cache.netease.com/cnews/2015/11/9/20151109114256148e6.jpg)


---


最近，俄罗斯客机在埃及西奈半岛坠毁后不久，伊斯兰国西奈分支就宣布对此负责，尽管目前真假难辨。似乎每逢发生恐怖袭击或重大灾难，都会有恐怖组织忙不迭地宣称，是他们干的。


恐怖分子认领恐怖袭击，相关研究不多，结论也各异。北伊利诺伊大学研究者艾伦·霍夫曼 ( Aaron M. Hoffman ) 研究发现，1968年至2004年，在以色列发生的356起恐怖袭击，200起有人认领。恐怖主义研究专家布鲁斯·霍夫曼  ( Bruce Hoffman ) 则称，二十世纪六七十年代，现代恐怖主义刚刚兴起之际，几乎所有的恐怖袭击都会被认领，到了八九十年代，认领现象逐渐走弱。普林斯顿大学的研究者奥斯汀·莱特 ( Austin L. Wright ) 研究了1998年至2004年间6,759件恐怖袭击，只有14.5%的恐袭被认领。而近年的媒体报道显示，恐怖袭击的认领现象愈趋热烈， 范围也不局限于自己的地盘，而是走向全球，大有恐怖组织辛迪加的趋势。比如今年十月，孟加拉国首都达卡发生恐怖袭击，伊斯兰国宣布对此负责。早一些的五月，美国得州卡通比赛现场发生恐怖袭击，伊斯兰国也宣布负责。


![](http://img2.cache.netease.com/cnews/2015/11/9/20151109082910dcfe5.jpg)


> 恐怖组织的实力无法通过正常渠道示人，时不时弄个大新闻，方便快捷，然后贴上自己的标签，既能给普通民众以威慑，又能让反恐的政府和组织见识实力。


恐怖组织宣布对某事负责不仅仅因为是他们干的，其背后有复杂的考量。炫耀实力是其中之一。恐怖组织的规模、装备等显示实力的标杆无法通过正常渠道示人，时不时弄个大新闻，方便快捷，然后贴上自己的标签，既能给普通民众以威慑，又能让反恐的政府和组织见识实力，何乐而不为。恐怖袭击的规模越大越能显示武力之强 。布鲁斯·霍夫曼在《抗击新型恐怖主义》(Countering the New Terrorism ) 中写道，“在他们（恐怖分子）看来，公众和媒体对恐怖主义暴力越来越习惯和脱敏了。相应地，恐怖分子觉得这驱使他们采取更加致命的手段去实现过去所达到的效果。”所以，大型的恐怖袭击除个别情况，基本都有组织宣布负责。普林斯顿大学的研究者奥斯汀·莱特分析了1998年至2004年间6,759件恐怖袭击，发现恐怖袭击的规模越大，有组织认领的可能性就越大。


![](http://img4.cache.netease.com/cnews/2015/11/9/20151109081157ea068.jpg)


人弹也是显示实力的重要武器，有人愿意为组织去死，说明这个组织的吸引力和向心力非同一般。据美国政治科学家罗伯特·佩普 ( Robert Pape ) 在其著作《拼命去赢：自杀式恐怖主义的战略逻辑》( Dying to Win: The Strategic Logic of Suicide Terrorism ) 中的统计，几乎所有的自杀式恐怖袭击都被恐怖组织认领。


但是，有的组织没有金刚钻，偏揽瓷器活，经常出现“碰瓷式”认领，难免被打脸——当然，都当恐怖分子了，不怕被打脸 。2011年挪威爆炸和枪击事件发生后，一个叫“全球圣战助手”的恐怖组织宣布对此负责，不过这个组织名气太小，压根没人听过，干出这么大事儿的可能性不大，后来真凶浮出水面，是独狼恐怖分子布雷维克，讽刺的是，他本人是反伊斯兰的。2014年，马航 HM370失踪后，有个叫“中国殉难者旅”的不知名恐怖组织称对马航 MH370的消失负责，但没人听说过这个组织，一眼就被人看穿是假的，最后不了了之。在艾伦·霍夫曼的研究里，356次恐怖袭击，30次有多个组织宣布负责，其中14次认领出现矛盾现象，也就是说，至少有14次冒领。


> 恐怖主义是行动宣传，时不时要刷下存在感；也是大型广告，认领是锦上添花。


宣传也是重要考量。恐怖主义本身就是一种宣传。美国政治评论家沃尔特·拉克  ( Walter Laqueur ) 在其著作《恐怖主义时代》( The Age of Terrorism ) 一针见血地指出，“如果恐怖主义是                       行动宣传
 ( propaganda by the deed )，那么一次恐怖袭击的成功则主要取决于收到的曝光率。” 恐怖组织是矛盾的存在。一方面，他们隐姓埋名，偷偷活动，防止被人一锅端，另一方面，他们又不甘寂寞，时不时要刷一下存在感。恐怖主义研究者加布里埃尔·魏曼 (Gabriel Weimann ) 研究发现，恐怖分子越来越以媒体为中心了，他们的行动以博取媒体关注为目的，通过媒体散播自己的主张。


恐怖袭击是广告， 不仅是为了震慑对手，也是为了吸引人加入，是让那些一心想“抛头颅洒热血”的年轻人下定决心加入组织的催化剂 。广告需要辨识度，否则会出现这样的情况：明明是基地组织干的，如果不认领，或许就被以为是伊斯兰国下的手。


> 恐怖组织之间的竞争不亚于任何残酷的商业斗争。首先，资源是一定的，多一个竞争对手，意味着自己少一份资源。其次，袭击的归属就不好计算，容易被其他组织冒领，“长他人志气，灭自己威风”是大忌。


竞争，也是恐怖组织竞相认领袭击的内在考量。美国经济学家艾伦·克罗伊格 ( Alan Krueger ) 在《何以成恐怖分子》 ( What Makes a Terrorist: Economics and the Roots of Terrorism )  一书中指出，恐怖主义是一个市场。有市场必有竞争。恐怖组织之间的竞争不亚于任何残酷的商业斗争。首先，同一地区，人口、资金甚至炸弹等资源都是有限且稀少的，多一个竞争对手，意味着自己少一份资源。 其次，如果某一地区只有一两个恐怖组织，尚且好办，但恐怖组织一多，袭击的归属就不好计算，得先下手为强，否则容易被其他组织冒领，“长他人志气，灭自己威风”是大忌。


认领也是在向金主投桃报李的方式：你们的钱没有白花，我们都用在刀刃了。中东地区的恐怖组织，幕后有金主支持的不在少数。金主有限而恐怖组织繁多，僧多粥少的情况下，只有通过优异的表现，才能成功吸引资助者的注意，获得赞助。


![](http://img2.cache.netease.com/cnews/2015/11/9/201511090845096729a.jpg)


比如2000年至2001年间，巴勒斯坦北撒玛利亚和杰宁的两支恐怖组织实施了九次报复性的自杀式炸弹袭击，两支恐怖组织竞相宣称是他们实施的袭击，其实有些并不是自己干的，目的是为了削弱对方的势力，在竞争中压过对方一头。艾伦·霍夫曼的研究验证了这一点，他发现，活跃的恐怖组织只有2个时，认领的几率只有24%，当数量上升至8个时，认领的几率则提高到了75%。


恐怖组织之间的竞争如此激烈，甚至会促进创新，托德·桑德勒 ( Todd Sandler ) 和丹尼尔· 阿尔塞 ( Daniel G Arce ) 在《恐怖主义：一种博弈论路径》 ( Terrorism: A Game-Theoretic Approach )中称，为了防止其他组织冒领，有些恐怖组织使用独特的袭击模式，让人一看便知是谁所为。但山寨不止中国人会，恐怖分子也能，所以更独家的防伪标志必不可少，认领袭击就是其中一个。


当然，原因远不止这些。比如，认领的成本低。网络兴起后，认领恐怖活动几乎零成本，只需要在 Twitter 、Facebook 上发一条信息，或在 YouTube 上传一个视频即可，无像以往，还得发给电视台。此外，还有研究发现，恐怖分子的认领与他们自身属性和所处国家状况有关，恐怖分子政治目的越强，越倾向于认领恐怖活动；而所处地区越是开明，媒介越是自由，则越倾向于认领。
